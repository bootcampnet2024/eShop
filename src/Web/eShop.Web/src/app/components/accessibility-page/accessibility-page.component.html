Accessibility

Compromisso com Acessibilidade

Trabalhamos continuamente para garantir que o site eShop seja acessível a todos os usuários, independentemente de suas capacidades físicas ou cognitivas. Implementamos práticas inclusivas para proporcionar uma experiência confortável e funcional para todos que visitem nosso site.

Recursos e Melhorias em Acessibilidade

Criamos páginas com alto contraste e fontes ajustáveis para facilitar a leitura para pessoas com baixa visão.

Navegação com Teclado

Nosso site pode ser totalmente navegado utilizando apenas o teclado, garantindo acessibilidade para quem não utiliza mouse.

Compatibilidade com Leitores de Tela

Todas as áreas do site foram otimizadas para funcionar com tecnologias como JAWS e NVDA. Provendo assim as necessidades que cada tecnologia necessita.

Como Garantimos a Acessibilidade

Treinamento contínuo para a equipe de design e desenvolvimento sobre práticas de acessibilidade. Testes periódicos para garantir acessibilidade necessaria. Coleta de feedback de terceiros para identificar e implementar melhorias contínuas.
